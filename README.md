# Playwright Demo for Knack challenge

## Prerequisites

-   Clone the Repository

-   Install dependencies

    `npm install`

- Install Playwright

    `npx playwright install --with-deps`

-   Declare env variables in `.env` file in root of repo

    `E2E_BUILER_USER`
    `E2E_BUILER_PASSWORD`
    `E2E_LIVEAPP_USER`
    `E2E_LIVEAPP_PASSWORD`
    `WAREHOUSE_URL`
    `LIVEAPP_URL`



## To run the tests

-   Execute tests

    `npx playwright test`

-   Enable debug mode

    `export PWDEBUG=1`
